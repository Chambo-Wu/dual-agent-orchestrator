## 3. 目标定义

`goal mode` 的目标不是替代当前 `job`，而是在 `job` 之上提供一个 durable supervisor。

建议层级：

```text
goal
  -> one or more jobs
    -> one or more workflows
      -> task runs
```

一句话概括：

> `goal` 负责长期推进与阶段检查，`job/workflow` 负责单次具体执行。
